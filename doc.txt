Cвой язык связей(описание дерева) вертикальные соединители.
	       ---*db-------
	       |           |
	    mysql      ---mssql---
               |       |         |
	    engine  engine2   engine3


*db[mysql, mssql];
mysql[engine]=mssql[engine2, engine3];

1. Получить уникальные имена элементов(mysql, mssql, engine2, engine3 ...)  регуляркой(*корень + [a-z,])
2. Получить уникальные сущности регуляркой(a-z[a-z,]$)
3. Сформировать отсортированный массив(от корня до низа)
4. При его переборе строить дерево из символов

1. Все в 1-ну строку
2. Сплит по ; (разделение на уровни)
3. Сплит по = (разделение уровней на сущности)
4. Сплит по , (разделение сущестностей на элементы)
